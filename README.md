# EHotel

## Technologies
- Java
- PostgreSQL
- HTML
- CSS
- Boostrap

## How to Run
1. Install PostgreSQL
2. Run database.sql
3. Configure ConnectionDB.java
4. Deploy on Tomcat
5. Open http://localhost:8080/project

## Features
- Search rooms
- Sort rooms
- Book rooms
- Manage hotels, chains, employees, customer, rooms
- View available rooms per area
- View total capacity per hotel
